# Perimeter Protocol — Reclaim the Floor

*The room is the product, not the rack. Clear the perimeter first. A tidy half-empty rack with a buried floor is still a mad workshop.*

**Protocol start date:** _______________  
**Decide tote hard deadline:** _______________ (default: end of this calendar month, or 30 days from start — pick one and write it on the tote)

---

## Physical rules (from the vision)

1. **Nothing lives on the floor within arm’s reach of the rack** except one labeled Decide tote.
2. **One work surface.** Bench empties when the project ends.
3. **Rack as appliance.** Front clear, rear walkable, blanking panels in empty U, airflow unblocked.
4. **Noise/heat = family constraints.** Gear yields to shared life.
5. **Visual calm.** Labels, consistent cables, inventory sheet on the door.

---

## Zones in the room

```text
                    [ door ]
                       |
        clear walkway  |  clear walkway
                       |
   [ bench — empty    ]|  [ 48U rack — appliance ]
   [ after project    ]|
                       |
        [ ONE Decide tote — dated ]
        [ optional Exit staging — leaving this week ]
```

No third pile. No “temporary” cardboard cities.

## Single Decide tote

| Field | Fill in |
|-------|---------|
| Tote label text | `DECIDE — decide by YYYY-MM-DD` |
| Hard decision date | _______________ |
| What may go in | Items from audit tagged Decide; mystery cables; “maybe” spares |
| What may **not** go in | Trash; known Exit items (those go to Exit staging); Keep items that belong in rack/bin |
| When date hits | Everything remaining → Exit path per [`05-gear-exit.md`](05-gear-exit.md). No extension without a written charter note. |

**Automatic rule:** After the date, indecision is a decision to Exit.

## Step-by-step clear (Days 1–14)

### Pass 1 — Trip hazards and trash (30–60 min)

- [ ] Bag obvious trash, food, dead batteries, shredded cardboard
- [ ] Open a path from door to rack front and rack rear
- [ ] Coil live power cables off the walking path

### Pass 2 — Sort floor into three only (60–120 min)

Using tags from [`02-inventory-audit.md`](02-inventory-audit.md):

- [ ] **Keep** → rack, labeled shelf, or labeled parts bin (bin must have a list)
- [ ] **Decide** → into the single Decide tote only
- [ ] **Exit** → Exit staging (one corner or one tote), scheduled to leave

If it does not fit: it is not Keep. Split, Exit, or put in Decide with the date.

### Pass 3 — Bench and surfaces

- [ ] Clear bench to bare top
- [ ] One project kit allowed only if named in the Lab Charter
- [ ] Tools hang or drawer — not rack-top sprawl

### Pass 4 — Rack face

- [ ] Remove gear already tagged Exit (power down safely first)
- [ ] Install blanking panels where U freed
- [ ] Affix door inventory sheet
- [ ] Confirm front clearance and intake/exhaust path

---

## Ongoing perimeter law

| Law | Enforcement |
|-----|-------------|
| New item enters room | Must have Keep home same day, or goes to Decide/Exit |
| Project ends | Bench empties same stewardship block |
| Decide tote overflows | Do not buy a second Decide tote — process Exit until it fits |
| Two stewardship skips | Shrink stack; re-clear perimeter before new gear |
